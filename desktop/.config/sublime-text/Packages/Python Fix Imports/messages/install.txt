Python Fix Imports
==================

Python Fix Imports is a plugin that helps you maintaining your imports statements in your
python file in a clean and efficient way.

Usage
-----

Key Bindings
************

The default key bindings for this plugin:

**Windows, Linux:**
+ `ctrl+alt+shift+i`: Fix imports in current file

**OSX:**
+ `command+ctrl+shift+i`: Fix imports in current file

Command Palette
***************

Open the command palette, it appears as ``Python Fix Imports``.
